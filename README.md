## Installation

Clone Repo:

```$ git clone https://github.com/Shaners/portfolio.git```

Install Dependencies:

```$ npm install gulp gulp-autoprefixer gulp-cssnano gulp-concat gulp-uglify gulp-notify gulp-rename --save-dev```

Or:

```$ npm install```
